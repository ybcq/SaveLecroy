## 力科示波器网线存图

力科最近也在进军低端示波器市场了。不过其部分示波器存图功能很不方便。故站长编写了该工具。

该工具用 python 编写，代码开源，需要事先安装驱动 ActiveDSO 2.36。

早期写的，和某公司的ScopeTools Pro使用的不是一套代码，无侵权相关风险。

内含aardio和python两种版本，按需下载。
